# IO 体系
&emsp;&emsp;Java 中为输入和输出提供了完善的API接口，可以方便地进行二进制文件或文件读写操作，此外也提供了对象序列化机制，对一个对象读写操作。

## 输入和输出流
&emsp;&emsp;Java IO体系上按照流的操作方式将流分为了输入流（`InputStream`）和输出流（`OutputStream`）。
