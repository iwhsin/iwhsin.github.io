# 1. JSR 规范请求

## 1.1. 常见 JSR 规范请求

| 规范请求编号                  | 背景描述                                                                   |
| ----------------------------- | -------------------------------------------------------------------------- |
| JSR-175（注解规范请求）       | Java 的元数据标记,允许标记类、接口、字段、方法等并使其具有特定的含义或属性 |
| JSR-269（可插拔注解处理 API） | 提供对 JSR-175 中注解提供 API，并为 Java 语言进行编程建模                  |
