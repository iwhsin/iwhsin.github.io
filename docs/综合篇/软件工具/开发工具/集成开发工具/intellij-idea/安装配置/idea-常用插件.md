# 1. IDEA 常用插件

## 1.1. 实用插件

&emsp;&emsp;汇总平时开发中比较实用的一些插件。

- **Maven Helper**<br>
  &emsp;&emsp;Maven 项目依赖分析和冲突解决。

- **Lombok**<br>
  &emsp;&emsp;通过注解的形式简化代码编写，如构造方法，getter、setter 方法等。

- **Rainbow Brackets**
  &emsp;&emsp;提供对称括号着色标记的功能。

- **SonarLint**

- aiXcoder

- RestfulTool

- Statistic

- Key Promoter X
