## Markdown基本语法

### 插入
* 插入图片：`![](图片绝对路径、相对路径或标准URL)`；
* 插入文件链接：[显示内容](文件绝对路径、相对路径、标准URL)。

### 表格
* 使用示例：
    ```
    | Header One     | Header Two     |
    | :------------- | :------------- |
    | Item One       | Item Two       |
    ```
* 对齐方式：`:`表示对齐方式，默认居中对齐。
    * `:----` 表示左对齐；
    * `----:` 表示右对齐；
    * `:----:`表示居中对齐。
